卡诺图和QM算法（非教程向）

2021-03-05

卡诺图化简法与Q-M化简法，我想了一周时间，还是没有比教材更好，更精简的讲法，所有的该奶奶都是必要的，所有的步骤都是必要的，环环相扣，不可或缺，既然如此，不如直接看教材好了，这里权作为笔记。

1. 布尔函数的化简

布尔代数可以用于描述组合逻辑的函数关系。组合逻辑是指一个无记忆的系统，我们向这个系统中输入一些值，立即可以得到一些输出。比如说y=p⋅s+q⋅sy=p·s+q·sy=p⋅s+q⋅s, 我们向系统中输入p, q, s 就可以得到输出y。

上面就是一个布尔函数的表达式，这个表达式是最简的吗？并不是，它可以变成y=(p+q)⋅sy=(p+q)·sy=(p+q)⋅s 这个式子只需要两步计算，而上面的式子需要3步。

由于我们的组合逻辑电路是用晶体管实现的，我们当然希望计算步骤越少越好，越少，我们需要的晶体管就越少，电路的性能就会更优越。

这时我们就需要一个程序性的，可以解决所有组合逻辑电路的布尔函数化简方法。

2. 卡诺图法和Q-M法

这是两个化简的方法，我们将通过一个例子来说明这两种方法。

2.1 卡诺图法

CD\AB 00 01 11 10
00 0 0 0 0
01 0 1 1 0
11 0 0 1 0
10 0 1 0 0

2.1.1 卡诺图法首先要列出一个真值表，这里列出了四变量真值表的列法

如果是三变量，是以下形式

C\AB 00 01 11 10
0 0 0 0 0
1 0 1 1 0

如果是二变量

B\A 0 1
0 1 0
1 1 1

四变量以上卡诺图是可以解决的，但不建议使用，最好使用后面要介绍的Q-M法

我们还回到4变量的例子。

卡诺图的列法中唯一需要注意的点是AB 或CD值的连续变化，即00→01→11→1000\rightarrow 01\rightarrow11\rightarrow1000→01→11→10 相邻两值之间仅有一位发生变化。

2.1.2 接下来我们要根据卡诺图写表达式，要用最大的矩形把所有的1项包围起来。

目标就是用最大最少的方框把所有的1都括起来，这时候，就有一些框是必不可少的，我们优先先把他们括起来，其他的再想办法，通过这种方法我们就能实现布尔函数的化简

2.3运算定律化简

有时候运算定律化简也是非常有必要的，这里写几个重要的化简定律

  1. 德摩根律

    (X+Y+Z)‾=X‾⋅Y‾⋅Z‾\overline{(X+Y+Z)}=\overline{X}·\overline{Y}·\overline{Z} (X+Y+Z)​=X⋅Y⋅Z

    X⋅Y⋅Z‾=X‾+Y‾+Z‾\overline{X·Y·Z}=\overline{X}+\overline{Y}+\overline{Z} X⋅Y⋅Z=X+Y+Z

    德摩根律可以这样表述：将变量因子用他们的逆替代，将·用+替代，用+将·替代，将0和1呼唤，就可以由原始表达式得到他的逆表达式。

    与之对应的有一个同样重要的对偶定律，但是这个对偶定律和布尔函数的化简没有关系了，他的作用是已知一个布尔函数律（比如德摩根律）是对的，那么可以通过对偶定理推出它的对偶同样是成立的，对偶表述为：and 和or互换，0和1互换，同时布尔变量保持不变。

    区别有两点，德摩根律得到的式子要画等号是要加¬\lnot¬的，而对偶本意就不是求等，即便加了等也不对，所以不加¬\lnot¬

    另一点是德摩根律要把变量求反，而对偶是不求反的。

  2. 互补律

    X+X‾=1X+\overline{X}=1 X+X=1

    X⋅X‾=1X·\overline{X}=1 X⋅X=1

  3. 化简准则

    X⋅Y+X⋅Y‾=X,(X+Y)⋅(X+Y‾)=XX·Y+X·\overline{Y}=X,\quad (X+Y)·(X+\overline{Y})=X X⋅Y+X⋅Y=X,(X+Y)⋅(X+Y)=X

    X+X⋅Y=X,X⋅(X+Y)=XX+X·Y=X,\quad X·(X+Y)=X X+X⋅Y=X,X⋅(X+Y)=X

    (X+Y‾)⋅Y=X⋅Y,X⋅Y‾+Y=X+Y(X+\overline{Y})·Y=X·Y,\quad X·\overline{Y}+Y=X+Y (X+Y)⋅Y=X⋅Y,X⋅Y+Y=X+Y

    X可以拆解上上面这么多种形式，这只能说明这不是一个解决问题的好方法

  4. 合并和因式分解定理

    (X+Y)⋅(X‾+Z)=X⋅Z+X‾⋅Y(X+Y)·(\overline{X}+Z)=X·Z+\overline{X}·Y(X+Y)⋅(X+Z)=X⋅Z+X⋅Y

    X⋅Y+X‾⋅Z=(X+Z)⋅(X‾+Y)X·Y+\overline{X}·Z=(X+Z)·(\overline{X}+Y)X⋅Y+X⋅Z=(X+Z)⋅(X+Y)

  5. 多数定律

    X⋅Y+Y⋅Z+X‾⋅Z=X⋅Y+X‾⋅ZX·Y+Y·Z+\overline{X}·Z=X·Y+\overline{X}·ZX⋅Y+Y⋅Z+X⋅Z=X⋅Y+X⋅Z

    (X+Y)⋅(Y+Z)⋅(X‾+Z)=(X+Y)⋅（X‾+Z）(X+Y)·(Y+Z)·(\overline{X}+Z)=(X+Y)·（\overline{X}+Z）(X+Y)⋅(Y+Z)⋅(X+Z)=(X+Y)⋅（X+Z）

3. 化简的结果

化简的最终目的是简化电路，使电路的性质更优越，我们上面讲了很多的电路化简方法，但一直没有解决什么样的电路才是最简电路的问题，而这个问题的缺失其实是很致命的。

电路的简单程度可以用晶体管的数目来衡量，下表列出了常见运算的晶体管数目

门电路 晶体管个数
非门 2
与非门 4
或非门 4
与门 6
或门 6
异或门(\lnot A\and B)\lor(A\land \lnot B) 16(是小于左边逻辑实现所需要的晶体管数的)
同或门(\lnot A\or B)\land(A\lor \lnot B) 16
