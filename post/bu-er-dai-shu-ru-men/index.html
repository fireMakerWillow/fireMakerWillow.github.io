【数逻1】布尔代数入门

2021-02-22

本文参考自布尔代数入门｜阮一峰

集合论用于表达人的逻辑思维

逻辑推导完全可以写成数学的形式，它更精确，使我们更容易得到隐晦的结果。逻辑推导用到的数学工具是集合。我们以

大前提：人都是会死的

小前提：苏格拉底是人

结论：苏格拉底会死

来展示用集合的语言描述逻辑。

我们定义集合R为所有人类，定义D为所有会死的东西，于是我们将大前提

人都是会死的

写成：

R X D = R

的形式，

我们定义S 为苏格拉底集合，小前提

苏格拉底是人

写成集合语言为

S X R = S

集合之间最基本的关系是并集和交集。这里用（X）来表示交集，用（+）来表示并集。

同样的，结论可以写成如下形式

苏格拉底会死

S X D = S

如何从大小前提出发得到结论呢？我们发现只要将大前提代入小前提即可

S X R

= S X (R X D)

= S X R X D

= S X D

= S

所以 三段论逻辑本质上是一种逻辑的迭代运算。

我们可以尝试着用得到的结论去解决一些更为复杂的逻辑运算。

大前提： 鸭嘴兽是卵生的哺乳动物。

小前提：鸭嘴兽是澳洲的动物

我们将描述转化为集合语言。定义Y为鸭嘴兽，R为卵生的哺乳动物，A为澳洲的动物。

大前提：Y X R = Y

小前提：Y X A = Y

代入运算

Y X R

= Y X （A X R）

= Y

我们得到了鸭嘴兽是卵生的澳大利亚哺乳动物的结论。这一结论也可以描述为有的卵生动物是澳洲的动物，或者有的澳洲动物是卵生的。

再尝试一个：

哲学家都是有逻辑头脑的

一个没有逻辑头脑的人总是很顽固的

P = 哲学家 L= 有逻辑头脑的人 S = 顽固的人，这里要用到‘非’的符号 -

P X L = P

(- L) X S = (- L)

这个需要非集的运算规则，最基本的运算规则需要定义空集和全集，空集定义为0，全集定义为I 则

A + (- A) = I

A X (- A) = 0

则

B

= B X I

= B X (A + -A )

= B X A + (- A) X B

有了这个式子做基础，我们来计算

P x (- L)

= P X L X (-L)

= P X 0

= 0

所以哲学家都不是没有逻辑的人成立

(-L) X S

= (-L) X S X (P + (-P) )

= (- L) X S X P + (- L) X S X (- P)

= (-L) X P X S + (- L) X S X (-P)

= (- L ) X S X (- P)

= (- L)

这个结论可以表述为 存在顽固的非哲学家，这些人中的一部分没有逻辑。

从集合论到布尔代数

通过上面的例子，我们意识到命题可以用集合表达，逻辑推导可以通过集合运算实现。

我们可以通过计算来证明命题的真伪。

比如下面一个问题

一名顾客进入了宠物店，对店员说：我要一个公猫，白色或黄色均可；或者一只母猫，除了白色，其他颜色都可以；或者只要是黑猫，我也要。

这时店员拿出一只灰色的公猫，请问是否满足要求。

我们当然可以继续用上面的方法来计算这个问题，

我们用 W，Y，B , G来表示白色，黄色，黑色和灰色

用 M 和 F 表示公猫和母猫，用C来表示顾客的要求，用S表示店员的供货。问题转化为

C = M X (W + Y) + F X (-W) + B

S = M X G

求 C X S

计算结果为

C X S

= M X (W + Y) X S + F X (- W) X S + B X S

= M X M X (W + Y) X G + F X M X (-W) X G + B X M X G

= 0 + 0 + 0

= 0

C X S 为空集，说明供货不满足要求

我们还有更简单的方式来解决这个问题，对于C中出现的每一个集合，我们用 1 表示S在该集合中，用 0 表是不在该集合中，得到

1 X (0 + 0) + 0 X 1 + 0

= 0

下面这个计算方法就是布尔代数。

布尔代数的运算法则

  1. 交集

1 X 1 = 1

1 X 0 = 0

0 X 0 = 0

  1. 并集

1 + 1 = 1

1 + 0 = 0

0 + 0 = 0

布尔代数的基本运算规则只有这些了。布尔代数实现了逻辑推理的数学化，程序化，能保证推理过程的正确。这是一个非常伟大的进步。
